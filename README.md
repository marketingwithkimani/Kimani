# Marketing with Kimani
**The African Marketing Authority**

AI-powered Relationship Engine and Strategy platform for trust-driven marketing.

## Features
- **Relationship AI Engine**: Interactive chat demo powered by Claude (Anthropic).
- **The 67% Framework**: Strategic lead capture systems focused on the silent majority of the market.
- **AI Command Center**: Dashboard for lead discovery, email automation, and relationship tracking.
- **Lead Discovery Crawler**: Autonomous research of high-potential contacts across industries.

## Tech Stack
- Frontend: HTML5, Vanilla CSS, JS
- Backend: Node.js (Express), AI Integration (Anthropic SDK)
- Automation: Nodemailer for SMTP delivery

## Setup
1. Clone the repository.
2. Install dependencies in the `Relationship/` directory.
   ```bash
   cd Relationship
   npm install
   ```
3. Set up your `.env` file in the `Relationship/` directory (Refer to `.env.example`).
4. Start the AI Brain:
   ```bash
   npx tsx execution/server.ts
   ```
5. Access the dashboard via `dashboard.html`.
